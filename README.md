# SOTECH - Logiciel de Gestion Web

## Description
SOTECH est un logiciel de gestion web développé en PHP pour les entreprises spécialisées dans la vente d’appareils Apple. Il permet une gestion complète et professionnelle des ventes, du stock et des opérations journalières.  

Ce logiciel inclut :  

- Gestion des articles avec **numéros de série**  
- Suivi des **entrées et sorties de stock**  
- Enregistrement et **historique complet des ventes** (comptant, crédit, multi-paiements)  
- Journalisation complète des opérations du logiciel  
- Gestion des clients et transactions
- vente à crédit 

SOTECH est conçu pour offrir un suivi précis et efficace, garantissant une meilleure organisation et un contrôle optimal de l’inventaire.

---

## Fonctionnalités principales

- **Gestion des stocks** : ajout, modification et suivi des articles avec numéros de série  
- **Ventes et transactions** : enregistrement des ventes comptant, à crédit et multi-paiements  
- **Historique complet** : journal des opérations avec détails sur chaque transaction  
- **Alertes et vérifications** : contrôle de cohérence des stocks et intégrité des données  
- **Interface Web intuitive** : accès simple et rapide aux modules de gestion  

---

## Technologies utilisées

- PHP pour le développement backend  
- MySQL pour la base de données  
- HTML, CSS et JavaScript pour l’interface web  
- MERISE pour la conception de la base de données  
- Notions de Laravel, Git, Docker (pour le déploiement et gestion de versions)
